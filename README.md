# MVP_Data_Analysis_Leadscoring_PUC
O presente projeto tem por objetivo analisar e explorar a base de dados para a construção de um modelo de classficação de leads entre qualificados e não qualificados. 

A partir disso, é possível sinalizar ao time comercial quais são aqueles de maior propensão, evitando que se percam durante a jornada de compra. 

Isso reduz o escopo de atuação dessas equipes, possibilitando ter um time mais enxuto, além de possibilitar a personalização da comunicação para cada grupo e além de ser possível estimar a receita até antes mesmo da abertura das inscrições.
